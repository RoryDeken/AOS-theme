# Description

aos is a simple and lightweight WordPress theme suitable for news, magazine, owned media and blog websites. It is fully responsive and mobile friendly, especially mobile horizontal scroll menu is very accessible. You can customize theme easily such as header logo, color, social links, etc. It supports a Schema.org microdata markup.

# テーマ説明

「aos」はシンプル＆軽量なWordPressテーマで、マガジン・ニュース・オウンドメディア・ブログなどのWebサイトに最適です。レスポンシブデザインかつモバイルフレンドリーで、モバイル環境での横スクロールメニューは視認性が高くアクセスが容易です。また、ヘッダーロゴ・色・ソーシャルリンクなどのカスタマイズを簡単に行うことができます。Schema.org + microdataによるマークアップを施しています。
